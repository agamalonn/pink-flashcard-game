# Pink Flashcard Game

Pink Flashcard Game is a desktop vocabulary-learning app built with Python and Tkinter. It was designed as a small but complete learning tool for practicing English vocabulary with translation, sound, and progress feedback.

The project combines a playful interface with practical learning mechanics, making it useful both as a personal study tool and as a portfolio example of building interactive Python applications.

## Project Focus

- English vocabulary practice
- Flashcard-style learning flow
- Real-time progress tracking
- Audio-assisted memorization
- Friendly desktop UI built in Python

## Key Features

- Pink-themed Tkinter interface
- English-to-Hebrew vocabulary support
- Flashcard interaction model for repeated practice
- Sound/audio feedback for a more engaging learning experience
- Progress tracking during the learning session
- Lightweight desktop app with no web server required

## Technical Skills Demonstrated

- Python desktop application development
- Tkinter UI structure and event handling
- State management for user progress
- Designing an educational workflow around repetition and feedback
- Building a focused tool for a real personal learning need
